# Spotify Track Popularity Prediction

## 📌 Overview

Spotify Track Popularity Prediction is a machine learning project developed to predict the popularity score of Spotify tracks using their audio features. The project applies data preprocessing, feature engineering and a regression model to estimate track popularity.

---

## 🚀 Features

- Data Preprocessing
- Feature Engineering
- Machine Learning Model
- Popularity Prediction
- Simple Web Interface using Flask

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Flask
- HTML


---

## 📊 Workflow

1. Load Spotify dataset
2. Clean and preprocess data
3. Train Machine Learning model
4. Save trained model
5. Predict track popularity using Flask interface
